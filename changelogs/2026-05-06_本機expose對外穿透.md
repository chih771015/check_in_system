## 變更摘要

### 做了什麼

1. **cloudflared 快速穿透腳本（`scripts/expose.sh`）**
   - 自動安裝 `cloudflared`（透過 Homebrew，僅第一次）
   - 啟動 Docker Compose（套用 expose override）
   - 等待服務就緒後建立 HTTPS 隧道，印出可分享的外部 URL
   - 搭配 `scripts/stop.sh` 停止所有 Docker 服務

2. **expose 專用 nginx 設定（`docker/nginx.expose.conf`）**
   - 加入 Cloudflare 真實 IP 透傳（`CF-Connecting-IP`）
   - 固定 `X-Forwarded-Proto: https`，避免後端誤判協定
   - 上傳大小限制調整為 20m

3. **Docker Compose override（`docker/docker-compose.expose.yml`）**
   - 僅覆蓋 `frontend` service 的 nginx conf 掛載
   - 其他服務（postgres、backend、jaeger）完整繼承主 compose

4. **使用說明文件（`docs/local-expose.md`）**
   - 涵蓋架構圖、前置需求、啟動步驟、停止方式、常見問題、安全注意事項

### 影響範圍

| 範圍 | 檔案 |
|------|------|
| 啟動腳本 | `scripts/expose.sh`、`scripts/stop.sh` |

### 注意事項

- **此分支不應合併到 `master`**，僅供本機對外測試使用
- cloudflared 免費快速隧道每次 URL 不同，需重新分享
- 啟動時發現 `Dockerfile.backend` 有 `COPY migrations` 的 build 錯誤，已修復並 cherry-pick 回 `master`
