## 變更摘要

### 做了什麼
病人管理新增 xlsx 批次匯入 / 匯出 / 範本（仿排班匯入與打卡匯出範式）：

- **匯入** `POST /api/admin/patients/import`：欄位「姓名、電話、證件類型(passport/hn/unid)、證件號碼」；逐列驗證 + 建立；**重複（證件類型+號碼已存在）或格式錯略過並回報**（`{created, skipped, errors:[{row, reason}]}`，row 為 sheet 列號），單列壞不影響整批。
- **匯出** `GET /api/admin/export/patients`：全部病人 xlsx，欄位與匯入相容（可再匯入）。
- **範本** `GET /api/admin/export/patients-template`：表頭 + 範例列。
- 前端 `PatientManagement` 工具列加「下載範本／匯入／匯出」；匯入完成顯示新增/略過筆數，略過列以 modal 列出列號與原因。

### 路由配置（避開 gin wildcard 衝突）
- import 放 **POST 樹**（`/patients/:id` 只在 PUT/DELETE/GET，POST 樹無 `:id`）。
- export / template 放 **`/export/*`**（GET 樹的 `/patients/:id/history` 佔了 `:id`，故不放 `/patients/export`）。
